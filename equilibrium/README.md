# Equilibrium — Mobile-First Layer-1 Blockchain

Native coin (EQU), Proof-of-Stationarity, Rust core, mobile mining.

## Build
- Rust: `cargo build --release`
- Android: `cargo ndk --target aarch64-linux-android build --release`
- iOS: `cargo build --target aarch64-apple-ios --release`

## Testnet
`cargo run --bin testnet-node`
