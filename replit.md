# Equilibrium

A Rust-based Layer-1 blockchain with Proof-of-Stationarity consensus, mobile mining, ZK proofs, libp2p P2P networking, and a full TypeScript node stack with a real-time block explorer, self-custody browser wallet, and WASM smart contracts.

> **Status (July 2026):** Mainnet-readiness hardening complete in Replit. **151 tests pass** (28 Rust, 123 TypeScript). Security audit fixes applied. Remote load test passed: **149 TPS sustained, p95 70 ms, 9,009/9,009 txs accepted** (50 VUs √ó 60 s, real HTTPS via Replit proxy). Remaining work is infra/ops (multi-region nodes, HA Postgres, monitoring, security audit) ‚Äî external to Replit.

## Run & Operate

- `pnpm --filter @workspace/api-server run dev` ‚Äî run the API node (port 8080, auto-mines every 15 s)
- `pnpm --filter @workspace/explorer run dev` ‚Äî run the block explorer + wallet (port 5000)
- `pnpm run typecheck` ‚Äî full typecheck across all packages
- `pnpm run build` ‚Äî typecheck + build all packages
- `pnpm --filter @workspace/api-spec run codegen` ‚Äî regenerate API hooks and Zod schemas from the OpenAPI spec
- `pnpm --filter @workspace/api-server run test` ‚Äî run 123 TypeScript tests
- `cd equilibrium && cargo test --lib` ‚Äî run 28 Rust tests
- Rust testnet: `cd equilibrium && cargo run --bin testnet-node`
- Rust wallet CLI: `cd equilibrium && cargo run --bin wallet`
- `pnpm --filter @workspace/coinomics run generate-genesis [outputPath]` ‚Äî write a fresh `genesis.json`

### After a container reset (recurring gotcha)

The Postgres workflow (`scripts/start-postgres.sh`) is fully idempotent ‚Äî it creates the runner role, pushes the schema as superuser, and grants table access on every boot. If the Postgres workflow started cleanly, the API Server workflow just works. If not, run:

```bash
bash scripts/start-postgres.sh
# then restart the API Server workflow
```

## Architecture decisions

- **Postgres-backed persistence** ‚Äî `blocks`, `transactions`, `validators`, `contracts` tables via Drizzle ORM. `initChain()` loads from Postgres on startup (falls back to genesis if empty). `scripts/start-postgres.sh` is idempotent and handles cold starts safely.
- **Contract-first API** ‚Äî OpenAPI spec lives in `lib/api-spec/openapi.yaml`; client hooks and Zod schemas are generated by Orval. Never hand-write API types in the client.
- **Address derivation** ‚Äî `SHA-256(raw_pubkey_bytes)[..20]` rendered as 40 hex chars. Matches Rust's `address_from_pubkey`. The explorer wallet hashes **raw bytes** (not the UTF-8 hex string) ‚Äî critical distinction; wrong encoding produces different addresses for the same keypair.
- **`@noble/ed25519` v3 API** ‚Äî use `ed.utils.randomSecretKey()` (not `randomPrivateKey`), `ed.etc.hexToBytes()` (not `ed.utils.hexToBytes`), and pass `Uint8Array` (not hex strings) to `signAsync`. All key ops use the async API (`getPublicKeyAsync`, `signAsync`).
- **Coinomics wired into live block production** ‚Äî `state.ts` imports `@workspace/coinomics` directly; `distributeBlockReward()`/`payValidatorReward()` split every coinbase and participation reward via `splitValidatorReward()`, and `slashValidator()` delegates to `applySlashing()`.
- **WASM VM validation** ‚Äî `WasmVM.deploy()` calls `WebAssembly.compile()` (not `validate()`) to reject invalid bytecode ‚Äî `compile()` throws, `validate()` only returns a boolean.
- **Fixed-point residuals** ‚Äî residuals are stored as `residualFp` (i64 scaled 1e18); `reorganize()` in `state.ts` uses BigInt comparison throughout; eliminates float non-determinism in fork choice. Rust `BlockHeader` uses `i64` for the same reason.

## Gotchas

- Do not run `pnpm dev` at workspace root ‚Äî run per-package with `--filter` or use the workflow buttons.
- `@noble/ed25519` v2 freezes `ed.etc` ‚Äî never assign `sha512Sync`; use async functions only.
- The Rust crate includes `libp2p` with `features = ["full"]` which downloads 470+ crates on first build ‚Äî expect `cargo build` to take several minutes on a cold cache.
- After editing `lib/api-spec/openapi.yaml`, always run codegen before touching client code: `pnpm --filter @workspace/api-spec run codegen`.
- Governance paths in openapi.yaml must go inside the `paths:` section (before `components:`), not appended to the end of the file.
- Rust unit tests: `cd equilibrium && cargo test --lib` (28 tests: wallet + stationary_solver + consensus).
- Load test: `k6 run scripts/load-test.js -e BASE_URL=https://<your-repl>.replit.dev` ‚Äî k6 binary must be re-downloaded after each container reset (not in PATH by default).
- `WebAssembly.compile()` throws on invalid WASM; `WebAssembly.validate()` only returns a boolean and never throws ‚Äî always use `compile()` for fail-fast rejection.
- Every container reset loses `node_modules`. Run `pnpm install` if needed, then restart all three workflows (Postgres ‚Üí API Server ‚Üí Explorer) in that order.
