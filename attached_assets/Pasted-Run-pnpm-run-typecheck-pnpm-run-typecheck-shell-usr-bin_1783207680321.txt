Step-by-Step Execution

1. Generate Validator Keypairs

If you don’t already have validator keypairs, run:

```bash
# For each validator (replace index 1..N)
cargo run --bin equilibrium keygen --output validator-1.key
cargo run --bin equilibrium keygen --output validator-2.key
# ... etc.
```

Then extract the addresses:

```bash
cargo run --bin equilibrium address --key validator-1.key > validator-1.addr
```

You’ll get a hex string like eq1abc.... Collect all validator addresses.

---

2. Create the Genesis Generation Script

Save this as scripts/generate-genesis.ts (if it doesn’t already exist):

```typescript
import { writeFileSync } from 'fs';

// === CONFIGURATION — EDIT THESE ===

const CHAIN_ID = "equilibrium-1";
const TIMESTAMP = new Date().toISOString(); // or a fixed date
const INITIAL_SUPPLY = "100,000,000 EQU";

const ALLOCATIONS = [
  { address: "eq1contrib...", amount: "20,000,000 EQU", vesting: "4-year linear", category: "core_contributors" },
  { address: "eq1backer...", amount: "15,000,000 EQU", vesting: "3-year linear", category: "early_backers" },
  { address: "eq1treasury...", amount: "25,000,000 EQU", vesting: "none", category: "community_treasury" },
  { address: "eq1mining...", amount: "30,000,000 EQU", vesting: "10-year linear", category: "mining_reserve" },
  { address: "eq1ecosystem...", amount: "10,000,000 EQU", vesting: "none", category: "ecosystem_fund" },
];

// Replace with your actual validator addresses and stakes
const INITIAL_VALIDATORS = [
  { address: "eq1val1...", stake: "500,000 EQU", name: "Validator One" },
  { address: "eq1val2...", stake: "500,000 EQU", name: "Validator Two" },
  // ... add all validators
];

const PARAMETERS = {
  target_block_time_ms: 15000,
  residual_threshold: 1e-7,
  initial_difficulty: 1_000_000,
  slashing_double_sign_pct: 5,
  slashing_downtime_pct: 1,
  unbonding_period_blocks: 10,
  max_validators: 100,
  governance_quorum_pct: 33.4,
  governance_voting_period_blocks: 10080,
};

// === GENERATE ===

const genesis = {
  chain_id: CHAIN_ID,
  timestamp: TIMESTAMP,
  initial_supply: INITIAL_SUPPLY,
  allocations: ALLOCATIONS,
  initial_validators: INITIAL_VALIDATORS,
  parameters: PARAMETERS,
};

writeFileSync('genesis.json', JSON.stringify(genesis, null, 2));
console.log('✅ genesis.json generated');
```

5. Place Genesis in the Chain

The API server (api-server) should load genesis.json on first start. Check artifacts/api-server/src/chain/init.ts (or similar) to ensure it reads from genesis.json. If not, you may need to wire it.

Typically, the chain will:

· Look for genesis.json at the project root or a known path.
· If the database is empty, it will load the genesis block and persist it.

You can test this by stopping Postgres, clearing the blocks table, and restarting the API server – it should create the genesis block from the file.
